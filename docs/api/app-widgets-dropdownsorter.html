Class app\widgets\DropdownSorter

All Classes | Properties | Methods | Events
Inheritanceapp\widgets\DropdownSorter » yii\base\Widget » yii\base\Component » yii\base\BaseObject
Implementsyii\base\Configurable, yii\base\ViewContextInterface

ButtonDropdownSorter renders bootstrap ButtonDropdown of sort links for the given sort definition.

ButtonDropdownSorter will generate a bootstrap ButtonDropdown which contains every attribute declared in $sort.
